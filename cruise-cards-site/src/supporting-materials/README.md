# Supporting Materials Templates

These templates are intended for PDF export (print to PDF from a browser or PDF renderer).

## Templates
- `pre-departure-checklist.html`
- `booking-summary.html`

## Branding
Update the CSS variables at the top of each template:
- `--brand-primary`
- `--brand-secondary`
- `--brand-accent`
- `--brand-text`

Replace the logo placeholder in the header with your logo URL.

## Export
Open the HTML file in a browser and "Print" -> "Save as PDF".
