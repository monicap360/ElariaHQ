# Master Content Cadence (Weekly)

This cadence assumes:
- News is reactive
- Authority is deliberate
- Feeder sites do the volume
- You approve everything before publish

## cruisesfromgalveston.net
Role: Decision Authority  
Cadence: 2–3 pieces / week (max)  
Quality > Quantity

Agents:
- Authority Editor Agent (primary)
- News Desk Agent (input only)
- Journal Agent (supporting depth)
- Experience Studio Agent (media + innovation tracks)

Weekly Output:
- Monday: News Desk -> Authority Editor (explainer on impact)
- Wednesday: Journal (evergreen advisory)
- Friday: Experience Studio Agent (video/doc + innovation package draft)
- Optional weekend: Authority Editor (seasonal or deployment context)

Examples:
- "Winter Storm Fern: What Galveston cruisers should actually do"
- "Carnival Breeze vs Jubilee from Galveston — who each ship is really for"
- "Why Galveston itineraries favor Cozumel & Costa Maya"
- "Ships arriving this week: mini documentary production brief"
- "Hybrid land + sea plans from Galveston: 1-night, 2-night, and post-cruise models"
- "Sustainable cruise tracking from Galveston with practical carbon offset options"

Rules:
- No deals
- No rates
- No urgency

## texascruiseport.com
Role: Regional Authority Feeder  
Cadence: 3–4 pieces / week

Agents:
- Publisher Agent
- Market Signal Agent

Weekly Output:
- Monday: Market Signal -> Publisher (port development / deployment news)
- Tuesday: Publisher (long-range outlook 2026–2028)
- Thursday: Publisher (infrastructure / capacity explainer)
- Optional Saturday: Publisher (regional comparison, factual only)

Examples:
- "What Carnival’s 2027/28 deployment means for Galveston capacity"
- "Why larger ships are rotating through Texas seasonally"

Notes:
- Feeds explanations upward to cruisesfromgalveston.net
- Not consumer-facing urgency

## houstoncruisetips.com
Role: Education & Prep  
Cadence: 2–3 pieces / week

Agents:
- Publisher Agent
- Journal Agent (lite mode)

Weekly Output:
- Tuesday: Publisher (prep guide)
- Thursday: Journal (deep-dive education)
- Optional Sunday: Publisher (FAQ-style explainer)

Examples:
- "Driving vs Flying to Galveston: what Houston travelers overlook"
- "How early should you arrive for a cruise from Texas?"

Notes:
- High-trust, low-competition SEO
- Funnels clarity -> authority site

## houstoncruiseshuttle.com
Role: Transportation Operations  
Cadence: Operational + event-based

Agents:
- Transportation Ops Agent
- Revenue Manager Agent
- Q&A Bot (logistics only)

Weekly Output:
- New sailing week: Revenue Manager (rate snapshot update)
- Weather/surge: Ops Agent (timing advisory)
- Continuous: Q&A Bot

Content Types:
- "Transportation from William P. Hobby Airport to Carnival Breeze"
- "Best pickup time if cruise check-in starts at 11:30am"
- "How luggage fits in a 4-passenger vehicle"

Language rules:
- Always "Transportation to/from"
- Never "port shuttle"
- Never "official"

## pier10parking.com & pier25parking.com
Role: Parking Capacity & Day-of Logistics  
Cadence: Event-driven + weekly refresh

Agents:
- Parking Ops Agent
- Publisher Agent (ops mode)

Weekly Output:
- Weekly: Ops Agent (capacity outlook)
- Cruise surge: Ops Agent ("What fills first" notice)
- As needed: Publisher Agent (timing & walking distance info)

Examples:
- "When Pier 25 parking typically fills on Jubilee sailings"
- "How early to arrive for Pier 10 parking on weekends"

## Social Media (Facebook / Events)
Role: Awareness, not conversion  
Cadence: 3–5 posts / week

Agent:
- Social Media Context Agent

Sources:
- News Desk
- Journal
- Authority drafts

Output:
- Calm summaries
- Facebook Events for sailing dates
- No prices
- No sales CTAs

## Q&A Bot (All Sites)
Role: Reduce repetitive questions  
Cadence: Continuous

Agent:
- Q&A Bot Agent

Scope:
- Transportation timing
- Pickup logistics
- Parking basics
- Cruise prep (not booking)

## Flow Overview
Google Alerts / News
  ↓
News Desk Agent
  ↓
Authority Editor (cruisesfromgalveston.net)
  ↓
Journal Agent (depth)
  ↓
Social Media Context Agent

Feeder sites run in parallel, not downstream.
